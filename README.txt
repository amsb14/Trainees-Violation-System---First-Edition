# المجلد يحتوي على ملفات ومجلدات متعلقة بالنظام:

1- ملف التشغيل (TVTC Violation Reporter) والذي يتم من خلاله عمل البرنامج.

2- مجلد (files) يحتوي على:

- ملف رايات (SP02.csv) والذي يحتوي على بيانات المتدربين وارقام جوالاتهم لجميع التخصصات
- ملف (Trainers_names_and_emails.csv) يحتوي على بيانات المدربين وايميلاتهم
- كما يحتوي على نموذج الكتابة/الطباعة الى ملف وورد (violation_temp.docx)والمستخدم من قبل البرنامج لتحرير المخالفات كما تم تعريفها بالحقول المحددة بالنموذج.

3- قاعدة البيانات (trainee_violations.db) والتي يتم إنشاؤها فور تشغيل البرنامج ويتم تسجيل جميع المخالفات إلى قاعدة بيانات ليتم إستدعائها عند الحاجة.

4- ملف النصي (violation_records.txt) ويتم إنشاْؤها عند استدعاء مدير النظام لسجل المخالفات المتعلق بالمتدرب. 

5- ملف (تعليمات.pdf) يحتوي على خطوات تعطيل نظام التنبيه المستمر ببرنامج البريد الالكتروني اوت لوك عندما يتم تشغيل البرنامج.

6- مجلد (icon) يحتوي على ملفات الايقونات (صور رمزية) يتم استخدامها من قبل البرنامج.
